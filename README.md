# Portal de Avisos Ciudadanos

## Descripción
Una plataforma diseñada para la gestión y visualización de notificaciones y avisos ciudadanos de manera eficiente para los usuarios.

## Stack Tecnológico
Este proyecto está construido utilizando las siguientes tecnologías:
- **React**
- **Tailwind CSS**
- **Firebase**

## Características principales
- **Sistema de alertas**: Notificaciones en tiempo real para mantener a los ciudadanos informados.
- **Modo de mantenimiento**: Función para desactivar temporalmente el acceso durante actualizaciones.
- **Aceptación de términos y condiciones**: Módulo integrado para asegurar el cumplimiento y acuerdo por parte de los usuarios.

## Instalación

Sigue estos pasos para levantar el proyecto en tu entorno local:

1. Instala las dependencias del proyecto abriendo tu terminal y ejecutando:
   ```bash
   npm install
   ```

2. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```

3. Abre tu navegador y navega a la dirección local que indique la consola (usualmente `http://localhost:5173`).
